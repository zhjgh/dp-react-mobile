# dp-react-mobile——基于react移动端UI组件库
dp-react-mobile是基于react的移动端UI组件库，目前不依赖任何第三方ui组件库，支持按需导入，可定制。

目前已开发完成如下ui组件：
* Slider 轮播图组件
* Ellipsis 省略组件


# 技术实现与版本
该组件库基于一下技术版本开发：
* react: 17.0.1
* react-dom: 17.0.1

## 使用

## 安装
``` js
npm install dp-react-mobile
```
或者用yarn安装
``` js
yarn add dp-react-mobile
```
## 使用
``` jsx
import { 
  Slider,
  Ellipsis,
} from 'dp-react-mobile'
```
